üîê Authentication

API menggunakan Simple Login untuk authentication, kemudian JWT (JSON Web Token) untuk session management.

Authentication Flow: 1. User memasukkan nomor telepon (dan optional: nama)
2. Backend check database
3. Jika user baru ‚Üí auto-register
4. Backend issue JWT token (valid 7 days)
5. Client store token dan gunakan untuk requests

üîë Authentication Endpoints

POST /api/auth/login Public

Simple login with phone number (auto-register if new)

Request Body:

{ "phoneNumber": "081234567890", "name": "John Doe" // Optional }

Success Response (200):

{ "status": "success", "message": "Login successful", "data": { "user": { "id": "65abc123...", "phoneNumber": "081234567890", "name": "John Doe", "role": "user", "isNewUser": false, "preferences": { ... } }, "token": "eyJhbGciOiJIUzI1NiIs..." } }
‚úÖ Auto-Register: Jika nomor telepon belum terdaftar, system otomatis membuat account baru dan langsung login.
GET /api/auth/profile Requires Auth

Get user profile

Headers:

Authorization: Bearer YOUR_TOKEN_HERE

Success Response (200):

{ "status": "success", "data": { "_id": "65abc...", "phoneNumber": "081234567890", "name": "John Doe", "isVerified": true, "preferences": { "usageType": "data", "budget": "medium" } } }
PUT /api/auth/profile Requires Auth

Update user profile

Request Body (all optional):

{ "name": "New Name", "profilePicture": "https://example.com/pic.jpg", "preferences": { "usageType": "data", "budget": "high", "interests": ["streaming", "gaming"] } }
POST /api/auth/check-phone Public

Check if phone number is registered

Request Body:

{ "phoneNumber": "081234567890" }

Success Response:

{ "status": "success", "message": "Phone number registered as John Doe", "data": { "available": false, "phoneNumber": "081234567890", "exists": true, "userName": "John Doe" } }
POST /api/auth/logout Requires Auth

User logout

üìù Note: Dengan JWT, logout handled di client-side dengan menghapus token dari localStorage/sessionStorage. Endpoint ini hanya untuk logging aktivitas logout.
GET /api/auth/users Admin Only

Get all users (Admin only)

Query Parameters:

Parameter Type Default
page number 1
limit number 20

üì¶ Product Endpoints

GET /api/products Public

Get all products with filtering and pagination
